Info Banner

Use

The info banner component is used to display messages on the front end to users, notifying the user when a request was completed or submitted without error.

Sample

  • This is an informational message.
  • You can add icons like the blue x above to clear the message.
  • That one doesn't work, but you can still click it.

<ign-info-banner> <button class="dismiss-button" type="button" aria-label="exit" mat-icon-button">×</button>
