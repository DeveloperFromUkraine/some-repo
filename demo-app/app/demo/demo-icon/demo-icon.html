Icon


Use

The icon component makes it easier to use vector-based icons. Material icons use geometric shapes to visually represent core ideas, capabilities, or topics.
See Material Design Style Icons and Angular Material Icon.


Sample

home
<mat-icon>home</mat-icon>
mat-icon matListIcon Creates icons to add to nav list items.
  • The mat-icon matListIcon component should be used for list items.
  • This component should not be used for selection lists.
  • This component should be used for nav lists.
