Expandable Search

Use

A search component which displays a search icon, which, when clicked, expands to display an input field with a button to go back to exit search and a button within the input field to clear search text.

Sample
<ign-expandable-search></ign-expandable-search>

Content

  • @Output() searchCriteria$: BehaviorSubject
    • Outputs search values for consumers to subscribe to

Style

  • The expandable search component should be displayed at the top of the encapsulating container.
