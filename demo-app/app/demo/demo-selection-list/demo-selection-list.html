Selection List

Use

A selection list provides an interface for selecting values, where each list item is an option. See Material Design List and Angular Material List Controls.

Sample

{{option}}

Options selected: {{options.selectedOptions.selected.length}}

<mat-selection-list #options>

    <mat-list-option *ngFor="let option of typesOfOptions">

        {option}

    </mat-list-option>

</mat-selection-list>

<p>

    Options selected: {{options.selectedOptions.selected.length}}

</p>
typesOfOptions = ['Option 1', 'Option 2', 'Option 3', 'Option 4'];
