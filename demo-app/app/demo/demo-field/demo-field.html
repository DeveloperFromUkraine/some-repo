Field


Use

This component displays a view field, consisting of a label and a value.


Sample

None editable Field:

42

Editable Field:

42

Editable Field with defined icon (home):

42

None editable Field:

<ign-field label="The Answer">42</ign-field>

Editable Field:

<ign-editable-field label="The Answer">42</ign-editable-field>

Editable Field with defined icon (home):

<ign-editable-field editIcon="home" label="The Answer">42</ign-editable-field>
