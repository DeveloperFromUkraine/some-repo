Angular Testing - Snapshot Testing

Snapshot Tests have to be deterministic

Set component data members that are in template to static values (ex. Date.now() would be different every run).

            
it('should match snapshot', async () => {{ '{' }}
    component.description = 'test description';
    Date.now = jest.fn(() => 1482363367071);

    await fixture.detectChanges();

    expect(fixture).toMatchSnapshot();
});

Now we can expect this date value to consistently return the same value every run.

</a>
