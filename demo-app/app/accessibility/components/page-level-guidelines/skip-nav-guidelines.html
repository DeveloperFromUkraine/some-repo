Skip Navigation

The following are page level, skip navigation guidelines should be adhered to when implementing new pages:

  • Why?

    • With skip navigation integration, keyboard users are able to skip to the main content element of a page without needing to click through every navigational element before it.

  • How?

    • Skip navigation should be able to be utilized either via browser extension or by activating the focused Skip Navigation link.

  • Code Examples

    • The skip navigation functionality has already been integrated into the UltiPro web appplication for page level accessibility and the ign-list-content component. For new applications, please utilize the below templates.

    • Page Level
      • 
                    <a id="skip-main" class="skip-main" href="javascript:void(0)" target="_self">Skip to main content</a>
                    <div id="body-container" role="main" tabindex="-1">
                        <router-outlet></router-outlet>
                    </div>
    • List Content
      • 
                    <a id="skip-list" class="skip-main" href="javascript:void(0)" target="_self">Jump to content</a>
                    <ng-content select="ign-list-selection"></ng-content>
                    <div id="list-content-skip" role="main" tabIndex="-1"><ng-content select="ign-list-content-panel"></ng-content></div>
